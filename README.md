# 📊 Tesla vs GameStop Stock Analysis

This project analyzes stock price and revenue data of Tesla and GameStop using Python.

## 🔧 Tools Used
- Python
- yfinance
- BeautifulSoup
- Pandas
- Matplotlib

## 📌 What I Did
- Extracted stock data using yfinance
- Collected revenue data via web scraping
- Cleaned and processed real-world data
- Visualized stock price vs revenue trends

## 📈 Key Insights
- Tesla shows strong growth in both revenue and stock price
- GameStop stock price surged in 2021 without similar revenue growth
